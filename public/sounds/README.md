# FocusDesk Sound Assets

This folder contains lightweight WAV placeholders used by the timer and ambient audio system:

- `start.wav`
- `break.wav`
- `complete.wav`
- `pause.wav`
- `tick.wav`
- `rain.wav`
- `deep-focus.wav`
- `white-noise.wav`
- `nature.wav`

They are audible demo assets generated for the HCI project, not silent placeholders. Replace them with polished audio files if needed, keeping the filenames stable or updating `hooks/use-audio.ts`.
